Chrome Extension Icon Generator

This tool helps you create icons for your Chrome extension:

  1. Each canvas below represents an icon size needed for your extension
  2. Click "Draw" buttons to fill each canvas with a blue square (placeholder icon)
  3. Click "Download" buttons to save each icon
  4. Move the downloaded PNG files to your extension's "icons" folder
